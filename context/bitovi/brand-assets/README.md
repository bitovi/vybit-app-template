# Bitovi Brand Assets

This directory contains official Bitovi brand assets including logos and mascot variations.

## Logo Variants

The Bitovi wordmark logo is available in three color schemes:

### Dark Green (Primary)
- `Logo-dark green.svg` / `.png` / `.eps`
- Primary brand color (teal/dark green #00333D)
- Use on light backgrounds

### Red-Orange
- `Logo-red-orange.svg` / `.png`
- Secondary brand color
- Use for accent or variety

### White
- `Logo-white.svg` / `.png` / `.eps`
- Use on dark backgrounds or photos

## Mascot Variants

The Bitovi mascot is a friendly character with headphones, available in multiple formats:

### 2D Versions
- **2D Dark Green**: `Mascot-2D dark green.eps`
- **Black**: `Mascot-black.svg` / `.png`
- **Dark Green**: `Mascot-dark green.svg` / `.png`

### 3D Versions
- **Large Uses**: `Mascot-3D large uses.eps` - Optimized for larger applications
- **Small Uses**: `Mascot-3D small uses.eps` - Optimized for smaller sizes

## File Formats

- **SVG**: Scalable vector format, ideal for web use
- **PNG**: Raster format with transparency
- **EPS**: Vector format for print and professional design software

## Usage Guidelines

- Use the dark green logo as the primary brand identifier
- Use the white logo on dark backgrounds for contrast
- The mascot represents approachability and technical expertise
- Maintain adequate clear space around logos and mascot
- Do not alter colors, proportions, or add effects to brand assets
